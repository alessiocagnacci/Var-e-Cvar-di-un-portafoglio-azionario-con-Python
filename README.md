# Var-e-Cvar-di-un-portafoglio-azionario-con-Python
*Calcolo della Value at Risk e del Conditional Value at Risk di un portafoglio di titoli a scelta su Yahoo Finance*

Ho deciso di usare python per calcolare questi due elementi in modo da valutare la perdita massima attesa per il portafagio entro un intervallo di confidenza (Var) e la perdita media negli scenari peggiori (CVar).
Per la seconda ho usato sia simulazione di Monte Carlo che le serie storiche.
